## Requirement

1. Java 17 is required for the sample app.

## Run the Sample App
1. Edit `values/config.xml` ➜ `pw_endpoint` to point to your Wallet Service endpoint.
3. Edit `values/config.xml` ➜ `pw_app_id` to fill in your `APP ID`
